# Contribuire a GroqShell

Grazie per il tuo interesse verso GroqShell.  
GroqShell è un progetto personale, sviluppato e mantenuto dal suo autore.  
Al momento non sono previste pull request esterne, ma idee, segnalazioni e suggerimenti tramite issue sono sempre ben accetti.

## Segnalazioni e suggerimenti
Se desideri:
- segnalare un comportamento inatteso,
- proporre un miglioramento,
- condividere un’idea o un caso d’uso interessante,

puoi aprire una issue: verrà letta con attenzione e valutata con piacere.

## Pull request
Le pull request non sono aperte al pubblico.  
Eventuali modifiche al codice vengono gestite direttamente dal maintainer.

## Licenza
Qualsiasi suggerimento o idea condivisa tramite issue potrà essere integrata nel progetto sotto licenza **GPLv3**.

Grazie per la comprensione e per l’interesse verso GroqShell.
